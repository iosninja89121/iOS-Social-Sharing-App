Method Documentation

- (NSDictionary *) getLinkAtLocation: (CGPoint)  location

Returns a dictionary of data about the link that it at the location. Returns nil if there is no link. A link dictionary contains the following keys: "linkType" a TDLinkType that identifies the type of link @"range" the range of the link within the label text @"link" the link text. This could be an URL, handle or hashtag depending on the linkType value
